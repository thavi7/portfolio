# 🌐 Personal Portfolio Website

A modern, fully responsive **personal portfolio website** designed to showcase projects, skills, and achievements with **smooth animations and optimized performance**.

---

## 🚀 Overview

This portfolio is built to create a strong impression on recruiters by combining **clean UI design** with **high-performance animations** using GSAP. It highlights projects, skills, and contact information in an engaging and interactive way.

---

## ✨ Features

* 🎨 **Modern & Responsive Design**
  Fully optimized for mobile, tablet, and desktop

* ⚡ **GSAP Animations**
  Smooth scroll-triggered animations for better user engagement

* 🧑‍💼 **Recruiter-Friendly Layout**
  Clean structure for easy navigation and readability

* 📂 **Project Showcase Section**
  Displays key projects with details

* 📞 **Contact Section**
  Easy way for recruiters to connect

* 🚀 **Optimized Performance**
  Animations implemented without affecting page load speed

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Styling:** Tailwind CSS
* **Animations:** GSAP (GreenSock Animation Platform)
